# Getting started

## 1. Clone project:

```sh
git clone https://github.com/michaelgichia/chucksfacts.git
```
## 2. Install dependencies:

```sh
npm install
```
or 
```sh
yarn
```
## 3. Development workflow

Run development:

```sh
npm run start
```
or 
```sh
yarn run start
```
